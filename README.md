# DecodeLabs Data Analytics Project 1 – Data Cleaning & Preparation

## Overview

This project focuses on cleaning and preparing raw data using Microsoft Excel and Power Query as part of the DecodeLabs Data Analytics Internship Training Program.

## Objectives

* Identify and handle missing values
* Remove duplicate records
* Correct inconsistent date formats
* Standardize data types and text formatting
* Validate data accuracy and integrity

## Tools Used

* Microsoft Excel
* Power Query

## Tasks Performed

* Replaced null/missing values
* Checked for duplicate IDs
* Fixed inconsistent date formats
* Standardized text and numerical formats
* Validated calculated totals

## Files Included

* Raw Dataset
* Cleaned Dataset
* Validation Report

## Skills Demonstrated

* Data Cleaning
* Data Preparation
* Data Validation
* Power Query
* Analytical Thinking

Successfully transformed a messy raw dataset into a clean, structured, and analysis-ready dataset.
